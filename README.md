# NumericalAnalysis
Numerical analysis methods implementation, such as Bisection method, Newton, Secant, ....

## Requirements 
- py_expression_eval
```
  pip install py_expression_eval --user 
```
- sympy

```
  pip install sympy --user 
```

## Installation
```
 > python main.py
  
```
